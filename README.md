# IBIT Test Assignment

Монорепо: `apps/web` (React + Vite), `apps/mobile` (React Native, без Expo), `packages/ui` — общий UI-пакет.

## Запуск

Node 24.13.0 (`nvm use`)

```
npm install
npm run web
npm run ios
npm run android
```

Для iOS на машине должен быть установлен CocoaPods (`pod install` вызывается автоматически при `npm run ios`)
